## Возникшие проблемы и их решения

### Проблема 1: Сложность BidirectionalBridge
**Решение**: Полный переход на архитектуру двух независимых потоков
- Простота и надёжность
- Отсутствие сложной синхронизации
- Естественная последовательность сообщений

### Проблема 2: JSONL файлы в подпапках
**Решение**: Рекурсивный поиск с сортировкой по времени модификации
```typescript
searchDir(claudeDir); // Рекурсивно ищем во всех подпапках
allFiles.sort((a, b) => b.mtime.getTime() - a.mtime.getTime()); // Самый свежий первый
```

### Проблема 3: Enter не срабатывает для длинных сообщений  
**Решение**: Динамическая пауза в зависимости от длины сообщения
```typescript
const delay = Math.max(500, message.length * 2); // Минимум 500ms, +2ms за символ
```
