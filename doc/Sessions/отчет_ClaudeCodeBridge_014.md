## 📝 КЛЮЧЕВЫЕ ОТКРЫТИЯ И РЕШЕНИЯ

### 💎 **Главное открытие: claude-code-chat**
- **Готовое решение** real-time двусторонней связи
- **child_process.spawn** с `--output-format stream-json`
- **Построчный JSON парсинг** для real-time обработки
- **Session management** с resume функциональностью
- **WSL support** для кроссплатформенности

### 🔍 **Второе открытие: Usage Monitor**
- **JSONL логи Claude Code** в `~/.claude/projects/**/*.jsonl`
- **Автоматическое сохранение** всех операций Claude Code
- **Полная история** токенов, стоимости, сессий
- **Session recovery** из исторических данных

### ⚡ **Архитектурное решение: 3-уровневая система**
1. **Real-time layer** - мгновенная двусторонняя связь
2. **Persistence layer** - история и восстановление сессий  
3. **Analytics layer** - мониторинг использования и статистика

## 🛠️ ТЕХНИЧЕСКИЕ ДЕТАЛИ ИССЛЕДОВАНИЯ

### **📊 Форматы вывода Claude Code:**
```
Task(Analyze Rails codebase structure)...
* postgres:query (MCP){sql: "SELECT..."}...
* Fetch(https://example.com/)...
Done (16 tool uses * 22.2k tokens * 41.8s)
```

### **🔧 Ключевые технические решения:**
- **Stream JSON парсинг:** построчная обработка вывода Claude Code
- **JSONL логи:** автоматические логи в `~/.claude/projects`
- **Session resume:** `--resume sessionId` для продолжения диалогов
- **Фильтрация служебной информации:** по типам сообщений

### **⚠️ Ограничения VS Code API:**
- `Terminal.onDidWriteData` **УДАЛЁН** из стабильного API
- Proposed API недоступен для публикуемых расширений
- Необходимы альтернативные подходы через child_process
