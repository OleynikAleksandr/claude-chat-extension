# Проект: Claude Chat Extension для VS Code
## План разработки с делегированием специалистам

---

## 📋 **КРАТКОЕ ОПИСАНИЕ ПРОЕКТА**

**Цель:** Создать расширение VS Code с чат-интерфейсом, которое отправляет сообщения в активный терминал с запущенным Claude Code CLI, автоматически нажимая Enter для выполнения команд.

**Архитектура:** 
```
Чат UI (Webview) → Extension Host → Active Terminal → Claude Code CLI
```
